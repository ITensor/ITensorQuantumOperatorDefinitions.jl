ITensorQuantumOperatorDefinitions.siteinds — Method
siteinds(f::Function, N::Integer; kwargs...)

Create an array of N physical site indices where the site type at site n is given by f(n) (f should return a string).

source
ITensorQuantumOperatorDefinitions.siteinds — Method
siteinds(d::Integer, N::Integer; kwargs...)

Create an array of N site indices, each of dimension d.

Keywords

  • addtags::String: additional tags to be added to all indices
source
ITensorQuantumOperatorDefinitions.siteinds — Method
siteinds(tag::String, N::Integer; kwargs...)

Create an array of N physical site indices of type tag. Keyword arguments can be used to specify quantum number conservation, see the space function corresponding to the site type tag for supported keyword arguments.

Example

N = 10
s = siteinds("S=1/2", N; conserve_qns=true)
